# @cpp.js/package-gdal-ios

The **iOS** build of the precompiled **GDAL** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-gdal](https://www.npmjs.com/package/@cpp.js/package-gdal)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

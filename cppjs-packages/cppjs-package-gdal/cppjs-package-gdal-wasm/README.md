# @cpp.js/package-gdal-wasm

The **WebAssembly** build of the precompiled **GDAL** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-gdal](https://www.npmjs.com/package/@cpp.js/package-gdal)**.

## Supported targets
- WebAssembly (`wasm32`) — single-threaded
- WebAssembly (`wasm32`) — multi-threaded (needs `SharedArrayBuffer`; serve with COOP + COEP headers)

# @cpp.js/package-webp-ios

The **iOS** build of the precompiled **WebP** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-webp](https://www.npmjs.com/package/@cpp.js/package-webp)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

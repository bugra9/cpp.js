# @cpp.js/package-curl-ios

The **iOS** build of the precompiled **libcurl** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-curl](https://www.npmjs.com/package/@cpp.js/package-curl)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

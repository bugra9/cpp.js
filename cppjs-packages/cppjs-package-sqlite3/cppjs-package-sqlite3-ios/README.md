# @cpp.js/package-sqlite3-ios

The **iOS** build of the precompiled **SQLite3** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-sqlite3](https://www.npmjs.com/package/@cpp.js/package-sqlite3)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

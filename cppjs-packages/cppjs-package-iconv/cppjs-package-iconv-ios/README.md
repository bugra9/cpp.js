# @cpp.js/package-iconv-ios

The **iOS** build of the precompiled **libiconv** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-iconv](https://www.npmjs.com/package/@cpp.js/package-iconv)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

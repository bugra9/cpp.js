# @cpp.js/package-zstd-ios

The **iOS** build of the precompiled **zstd** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-zstd](https://www.npmjs.com/package/@cpp.js/package-zstd)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

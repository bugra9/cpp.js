# @cpp.js/package-openssl-ios

The **iOS** build of the precompiled **OpenSSL** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-openssl](https://www.npmjs.com/package/@cpp.js/package-openssl)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

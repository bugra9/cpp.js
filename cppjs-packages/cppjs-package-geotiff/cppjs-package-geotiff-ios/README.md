# @cpp.js/package-geotiff-ios

The **iOS** build of the precompiled **libgeotiff** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-geotiff](https://www.npmjs.com/package/@cpp.js/package-geotiff)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

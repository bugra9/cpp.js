# @cpp.js/package-geotiff-wasm

The **WebAssembly** build of the precompiled **libgeotiff** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-geotiff](https://www.npmjs.com/package/@cpp.js/package-geotiff)**.

## Supported targets
- WebAssembly (`wasm32`) — single-threaded
- WebAssembly (`wasm32`) — multi-threaded (needs `SharedArrayBuffer`; serve with COOP + COEP headers)

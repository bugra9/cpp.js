# @cpp.js/package-jpegturbo-ios

The **iOS** build of the precompiled **libjpeg-turbo** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-jpegturbo](https://www.npmjs.com/package/@cpp.js/package-jpegturbo)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

# @cpp.js/package-proj-ios

The **iOS** build of the precompiled **PROJ** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-proj](https://www.npmjs.com/package/@cpp.js/package-proj)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

# @cpp.js/package-geos-ios

The **iOS** build of the precompiled **GEOS** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-geos](https://www.npmjs.com/package/@cpp.js/package-geos)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

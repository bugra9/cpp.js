# @cpp.js/package-lerc-ios

The **iOS** build of the precompiled **LERC** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-lerc](https://www.npmjs.com/package/@cpp.js/package-lerc)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

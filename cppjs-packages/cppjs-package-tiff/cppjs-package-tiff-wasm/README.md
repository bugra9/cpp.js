# @cpp.js/package-tiff-wasm

The **WebAssembly** build of the precompiled **TIFF** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-tiff](https://www.npmjs.com/package/@cpp.js/package-tiff)**.

## Supported targets
- WebAssembly (`wasm32`) — single-threaded
- WebAssembly (`wasm32`) — multi-threaded (needs `SharedArrayBuffer`; serve with COOP + COEP headers)

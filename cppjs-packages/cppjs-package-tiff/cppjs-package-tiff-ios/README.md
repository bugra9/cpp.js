# @cpp.js/package-tiff-ios

The **iOS** build of the precompiled **TIFF** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-tiff](https://www.npmjs.com/package/@cpp.js/package-tiff)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

# @cpp.js/package-expat-android

The **Android** build of the precompiled **Expat** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-expat](https://www.npmjs.com/package/@cpp.js/package-expat)**.

## Supported targets
- Android `arm64-v8a` — 64-bit ARM (physical devices)
- Android `x86_64` — 64-bit (emulators)

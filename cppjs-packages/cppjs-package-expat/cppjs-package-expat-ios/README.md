# @cpp.js/package-expat-ios

The **iOS** build of the precompiled **Expat** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-expat](https://www.npmjs.com/package/@cpp.js/package-expat)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

# @cpp.js/package-zlib-wasm

The **WebAssembly** build of the precompiled **zlib** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-zlib](https://www.npmjs.com/package/@cpp.js/package-zlib)**.

## Supported targets
- WebAssembly (`wasm32`) — single-threaded
- WebAssembly (`wasm32`) — multi-threaded (needs `SharedArrayBuffer`; serve with COOP + COEP headers)

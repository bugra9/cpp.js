# @cpp.js/package-zlib-ios

The **iOS** build of the precompiled **zlib** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-zlib](https://www.npmjs.com/package/@cpp.js/package-zlib)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

# @cpp.js/package-spatialite-ios

The **iOS** build of the precompiled **SpatiaLite** library, built with [cpp.js](https://cpp.js.org).

> For installation and usage, see the main package: **[@cpp.js/package-spatialite](https://www.npmjs.com/package/@cpp.js/package-spatialite)**.

## Supported targets
- iOS device — `arm64`
- iOS Simulator — `arm64` (Apple Silicon)

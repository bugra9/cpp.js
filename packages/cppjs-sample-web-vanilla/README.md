# Getting Started

>**Note**: Make sure you have completed the [Cpp.js - Prerequisites](https://cpp.js.org/docs/guide/getting-started/prerequisites) instructions.

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Build native code to webassembly:

```bash
pnpm run build
```
